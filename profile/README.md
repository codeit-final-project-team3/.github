## 서비스 소개

- 즐거움을 나누는 이색 체험 플랫폼, GIVEMEFIVE에서 간단하게 여행을 예약해보세요!

## 핵심 기능

### 유저기능

1. 로그인 & 회원가입

2. 소셜 로그인

3. 개인페이지

### 체험 관리

1. 체험 등록

2. 체험 예약

3. 체험 취소

4. 체험 후기 작성
